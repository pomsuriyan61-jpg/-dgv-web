==================================================================
 DGV Investment Analyzer — ชุดสมบูรณ์ (เว็บ + หลังบ้าน)
==================================================================

ในโฟลเดอร์นี้มี:
  index.html                          ← หน้าเว็บ (เปลี่ยนชื่อไม่ได้ ต้องชื่อนี้)
  netlify.toml                        ← ไฟล์ตั้งค่า Netlify
  netlify/functions/chat.js           ← หลังบ้าน AI แชท (ซ่อนคีย์ + ค้นข่าวจริง)
  netlify/functions/news-forecast.js  ← หลังบ้านค้นข่าวมาปรับกราฟพยากรณ์

หลังบ้านนี้ทำให้:
  - คีย์ AI ซ่อนในเซิร์ฟเวอร์ ไม่มีใครเห็น (ปลอดภัย)
  - ทุกคนที่เปิดเว็บใช้ AI ได้เลย ไม่ต้องมีคีย์ตัวเอง
  - AI ค้นข่าวจริงระดับโลกมาวิเคราะห์/ปรับเส้นพยากรณ์ได้

==================================================================
 ขั้นที่ 0 — ยกเลิกคีย์เก่า สร้างคีย์ใหม่ (สำคัญ!)
==================================================================
ถ้าเคยวางคีย์ในแชต/ที่สาธารณะ ให้ยกเลิกก่อน:
  console.anthropic.com → API Keys → ลบคีย์เก่า → Create Key → เก็บคีย์ใหม่ไว้

==================================================================
 ขั้นที่ 1 — เอาไฟล์ขึ้น GitHub
==================================================================
วิธีง่ายสุด (ใช้ repo เดิมที่มีอยู่ หรือสร้างใหม่):
  1. เข้า github.com → repo ของคุณ
  2. กด "Add file" → "Upload files"
  3. ลากไฟล์ index.html, netlify.toml ขึ้นไป
  4. สำหรับโฟลเดอร์ netlify/functions ให้สร้างโครงสร้างโฟลเดอร์
     - ตอนอัปโหลด ลากทั้งโฟลเดอร์ "netlify" เข้าไป GitHub จะเก็บโครงสร้างให้เอง
     - หรือพิมพ์ชื่อไฟล์เป็น  netlify/functions/chat.js  ตอนสร้างไฟล์ใหม่ (GitHub จะสร้างโฟลเดอร์ให้)
  5. Commit

(ถ้าใช้คอม: ลากทั้งโฟลเดอร์ขึ้น GitHub Desktop หรือ git push ทีเดียวง่ายกว่า)

==================================================================
 ขั้นที่ 2 — เชื่อม Netlify กับ GitHub (สำคัญ — อย่าใช้ลากวางไฟล์)
==================================================================
หลังบ้าน (Functions) ต้องให้ Netlify build จาก GitHub เท่านั้น
การลากวางไฟล์เฉย ๆ จะไม่รันหลังบ้าน!

  1. เข้า app.netlify.com → "Add new project" (หรือ "Add new site")
  2. เลือก "Import an existing project" → "Deploy with GitHub"
  3. เลือก repo ของคุณ
  4. หน้า build settings ปล่อยค่าว่างได้เลย (netlify.toml จัดการให้แล้ว)
     - Publish directory: .  (จุด)
     - Functions directory: netlify/functions (อ่านจาก netlify.toml อัตโนมัติ)
  5. กด "Deploy"

==================================================================
 ขั้นที่ 3 — ใส่คีย์ AI เป็น Environment Variable (ซ่อนปลอดภัย)
==================================================================
  1. ใน Netlify เข้า project → "Site configuration" → "Environment variables"
  2. กด "Add a variable" → "Add a single variable"
       Key   :  ANTHROPIC_API_KEY
       Value :  (คีย์ใหม่ที่สร้างจากขั้นที่ 0 — ขึ้นต้น sk-ant-...)
  3. (ถ้าต้องการเปลี่ยนรุ่นโมเดล) เพิ่มอีกตัว:
       Key   :  ANTHROPIC_MODEL
       Value :  claude-sonnet-4-6   (หรือรุ่นที่บัญชีคุณใช้ได้)
  4. Save
  5. ไปที่แท็บ "Deploys" → กด "Trigger deploy" → "Deploy site"
     (ต้อง deploy ใหม่หลังใส่คีย์ คีย์จึงจะมีผล)

==================================================================
 ขั้นที่ 4 — เสร็จ! ทดสอบ
==================================================================
  - เปิดลิงก์เว็บ (เช่น your-name.netlify.app)
  - ไปหน้า "ถาม AI" → พิมพ์ "มีข่าวอะไรล่าสุดเกี่ยวกับ Apple" → AI จะค้นข่าวจริงมาตอบ
  - หน้า "กราฟ" → กดปุ่ม "🔎 ดึงข่าวจริง" ใต้กราฟพยากรณ์ → เส้นคาดการณ์จะปรับตามข่าวจริง
  - ทุกคนที่เปิดลิงก์นี้ใช้ได้เลย ไม่ต้องใส่คีย์

==================================================================
 ตั้งชื่อเว็บให้สวย
==================================================================
  Netlify → project → "Site configuration" → "Change site name"
  ตั้งเป็น dgv-analyzer → ได้ลิงก์ dgv-analyzer.netlify.app
  (อยากได้โดเมนของตัวเอง เช่น dgv.app ต้องซื้อโดเมนแล้วผูกใน Domain management)

  - Finnhub (ราคาสด) ฝังคีย์ไว้ในเว็บแล้ว ใช้ได้เลย (read-only ฟรี)
